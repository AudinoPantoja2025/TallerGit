Guia para HTML y CSS

Un ejemplo para empezar a practicar.

Qué incluye

Este ejemplo muestra:

  • Encabezado con degradado
  • Diseño en tarjetas (card)
  • Encabezado y pie de página

Cómo usarlo

Copia todo el contenido en un archivo index.html y ábrelo en tu navegador. Modifica texto y estilos para aprender.

Consejo

Cambia colores en el archivo styles.css para experimentar.
